# Cyborg Landing Page

A futuristic and responsive landing page inspired by cybernetic technology and AI-driven aesthetics. The website features a modern dark-themed interface with neon accents, animated elements, glassmorphism effects, and smooth user interactions.

## Features

* Fully responsive design
* Futuristic cyborg-inspired UI
* Smooth animations and transitions
* Modern typography and layout
* Interactive navigation
* Optimized for desktop, tablet, and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Design Techniques
* CSS Animations & Effects
