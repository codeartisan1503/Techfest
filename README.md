# Techfest

TECHFEST 2026 – Digital Dimension is an immersive 3D interactive event website built using Three.js, GSAP, HTML, CSS, and JavaScript. The website transforms traditional event promotion into a futuristic digital experience through cinematic scroll animations, interactive 3D objects, dynamic transitions, holographic UI elements, and real-time visual effects. Visitors explore multiple technology-themed zones including AI, Robotics, Coding, and Innovation while engaging with immersive animations and interactive environments. The project demonstrates modern web development, creative storytelling, 3D visualization, and user experience design.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
